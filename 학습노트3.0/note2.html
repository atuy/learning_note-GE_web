div
{
font : normal 24px verdata;
}
.dropText
{
text-shadow : 3px 3px; 그림자 위치를 지정함
}
.redText
{
text-shadow : 3px 3px red; 그림자 위치와 색상을 지정함
}
.blurText
{
text-shadow : 3px 3px 5px skyBlue; 그림자 위치와 번짐효과를 넣고 색상을 지정함
}
.glowEffect
{
text-shadow : 0px 0px 3px red; 번짐효과를 넣고 색상을 지정함
}
.wordArtEffect
{
color : white; 글자의 색상을 지정함
text-shadow : 0px 0px 3px darkBlue; 번짐효과와 색상을 지정함
}
.threeDEffect
{
color : white;
text-shadow : 2px 2px 4px black;그림자 위치와 번짐효과를 넣고 색상을 지정함
}
.multiEffect
{
color : yellow;
text-shadow : 2px 2px 2px black,
0 0 25px blue,

}
